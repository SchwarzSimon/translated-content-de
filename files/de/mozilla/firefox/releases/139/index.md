## Experimentelle Web-Features

Diese Features sind neu in Firefox 139 enthalten, aber standardmäßig deaktiviert.
Um sie auszuprobieren, suchen Sie die entsprechende Einstellung auf der `about:config`-Seite und setzen Sie diese auf `true`.
Weitere solche Features finden Sie auf der Seite für [Experimentelle Features](/de/docs/Mozilla/Firefox/Experimental_features).
