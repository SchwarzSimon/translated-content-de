## Variablen 1

Bei dieser Aufgabe möchten wir, dass Sie:

- Eine Variable namens `myName` deklarieren.
- `myName` mit einem geeigneten Wert initialisieren, auf einer separaten Zeile (Sie können Ihren tatsächlichen Namen oder etwas anderes verwenden).
- Eine Variable namens `myAge` deklarieren und sie auf derselben Zeile initialisieren.
