## Variablen 1

In dieser Aufgabe möchten wir, dass Sie:

- Eine Variable namens `myName` deklarieren.
- `myName` mit einem geeigneten Wert initialisieren, in einer separaten Zeile (Sie können Ihren tatsächlichen Namen oder etwas anderes verwenden).
- Eine Variable namens `myAge` deklarieren und initialisieren, in derselben Zeile.
