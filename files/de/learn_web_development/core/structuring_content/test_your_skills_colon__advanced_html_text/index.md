Das Ziel dieses Fähigkeits-Tests ist es zu bewerten, ob Sie verstehen, wie man [weniger bekannte HTML-Elemente für die Auszeichnung fortgeschrittener semantischer Merkmale verwendet](/de/docs/Learn_web_development/Core/Structuring_content/Advanced_text_features).

> [!NOTE]
> Sie können Lösungen in den interaktiven Editoren auf dieser Seite ausprobieren oder in einem Online-Editor wie [CodePen](https://codepen.io/), [JSFiddle](https://jsfiddle.net/) oder [Glitch](https://glitch.com/).
>
> Wenn Sie nicht weiterkommen, können Sie sich in einem unserer [Kommunikationskanäle](/de/docs/MDN/Community/Communication_channels) an uns wenden.

## Aufgabe 1

In dieser Aufgabe möchten wir, dass Sie der bereitgestellten HTML-Datei einige semantische Merkmale hinzufügen:

- Wandeln Sie den zweiten Absatz in ein Blockzitat um und kennzeichnen Sie semantisch, dass das Zitat aus [Accessibility](/de/docs/Learn_web_development/Core/Accessibility) stammt.
- Kennzeichnen Sie "HTML" und "CSS" semantisch als Akronyme und geben Sie Erweiterungen als Tooltips an.
- Verwenden Sie Tief- und Hochstellungen, um die korrekte Semantik für chemische Formeln und Daten anzugeben und diese korrekt darzustellen.
- Verbinden Sie maschinenlesbare Daten mit den Daten im Text semantisch.
