Ein **Boolean** in JavaScript ist ein {{Glossary("Primitive", "Primitivwert")}}, der entweder `true` oder `false` sein kann.

Ein {{jsxref("Boolean")}}-Objekt ist ein {{Glossary("wrapper", "Wrapper")}} um ein Boolean-Primitiv.
