{{HTMLSidebar("HTML_Elements")}}

Diese Seite listet alle {{Glossary("HTML", "HTML")}} {{Glossary("Element", "Elemente")}} auf, die mit {{Glossary("Tag", "Tags")}} erstellt werden.

Sie sind nach Funktion gruppiert, um Ihnen zu helfen, das Gewünschte leicht zu finden. Eine alphabetische Liste aller Elemente ist in der Seitenleiste auf jeder Element-Seite sowie auf dieser Seite vorhanden.

> [!NOTE]
> Für weitere Informationen über die Grundlagen von HTML-Elementen und -Attributen siehe [Anatomie eines HTML-Elements](/de/docs/Learn_web_development/Core/Structuring_content/Basic_HTML_syntax#anatomy_of_an_html_element).

## Hauptwurzeln

| Element                 | Beschreibung                                                                                                                                                                |
| ----------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| {{HTMLElement("html")}} | Repräsentiert die Wurzel (Top-Level-Element) eines HTML-Dokuments und wird daher auch als _Wurzelelement_ bezeichnet. Alle anderen Elemente müssen Nachkommen dieses Elements sein. |

## Metadaten des Dokuments

Metadaten enthalten Informationen über die Seite. Dazu gehören Informationen über Stile, Skripte und Daten, die Software ({{Glossary("search_engine", "Suchmaschinen")}}, {{Glossary("Browser", "Browser")}}, usw.) helfen, die Seite zu verwenden und darzustellen. Metadaten für Stile und Skripte können direkt auf der Seite definiert oder mit einer Datei verknüpft werden, die diese Informationen enthält.

| Element                  | Beschreibung                                                                                                                                                                                                                                                                                  |
| ------------------------ | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| {{HTMLElement("base")}}  | Gibt die Basis-URL an, die für alle relativen URLs in einem Dokument verwendet werden soll. Es kann nur ein solches Element in einem Dokument geben.                                                                                                                                           |
| {{HTMLElement("head")}}  | Enthält maschinenlesbare Informationen (Metadaten) über das Dokument, wie dessen [Titel](/de/docs/Web/HTML/Reference/Elements/title), [Skripte](/de/docs/Web/HTML/Reference/Elements/script) und [Stylesheets](/de/docs/Web/HTML/Reference/Elements/style).                         |
| {{HTMLElement("link")}}  | Gibt Beziehungen zwischen dem aktuellen Dokument und einer externen Ressource an. Dieses Element wird meistens verwendet, um auf CSS zu verweisen, kann aber auch verwendet werden, um Website-Icons zu etablieren (sowohl "Favicon"-Stil-Icons als auch Icons für den Homescreen und Apps auf mobilen Geräten), unter anderem. |
| {{HTMLElement("meta")}}  | Repräsentiert {{Glossary("Metadata", "Metadaten")}}, die nicht durch andere HTML-meta-bezogene Elemente wie {{HTMLElement("base")}}, {{HTMLElement("link")}}, {{HTMLElement("script")}}, {{HTMLElement("style")}} und {{HTMLElement("title")}} dargestellt werden können.              |
| {{HTMLElement("style")}} | Enthält Stilinformationen für ein Dokument oder einen Teil eines Dokuments. Es enthält CSS, das auf den Inhalt des Dokuments angewendet wird, das dieses Element enthält.                                                                                                                     |
| {{HTMLElement("title")}} | Definiert den Titel des Dokuments, der in der Titelleiste eines {{Glossary("Browser", "Browsers")}} oder im Tab einer Seite angezeigt wird. Es enthält nur Text; HTML-Tags innerhalb des Elements, falls vorhanden, werden ebenfalls als reiner Text behandelt.                              |

## Bereichswurzel

| Element                 | Beschreibung                                                                                     |
| ----------------------- | ----------------------------------------------------------------------------------------------- |
| {{HTMLElement("body")}} | Repräsentiert den Inhalt eines HTML-Dokuments. Es kann nur ein solches Element in einem Dokument geben. |

## Inhaltsgliederung

Inhaltsgliederungselemente ermöglichen es Ihnen, den Dokumentinhalt in logische Abschnitte zu organisieren. Verwenden Sie die Abschnittselemente, um eine grobe Gliederung für den Seiteninhalt zu erstellen, einschließlich Header- und Footer-Navigation, und Überschriftselemente, um Abschnitte von Inhalten zu identifizieren.

| Element                                                                                                                                                                                                                                                                                                                                                                                                                 | Beschreibung                                                                                                                                                                                                                                                                                                                                                                    |
| ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| {{HTMLElement("address")}}                                                                                                                                                                                                                                                                                                                                                                                              | Gibt an, dass die eingeschlossene HTML-Kontaktinformationen für eine Person oder Personen oder für eine Organisation bereitstellt.                                                                                                                                                                                                                                               |
| {{HTMLElement("article")}}                                                                                                                                                                                                                                                                                                                                                                                              | Repräsentiert eine eigenständige Komposition in einem Dokument, einer Seite, einer Anwendung oder einer Website, die unabhängig verteilt oder wiederverwendet werden soll (z. B. in Syndikation). Beispiele sind ein Forenbeitrag, ein Magazin- oder Zeitungsartikel, ein Blogeintrag, eine Produktkarte, ein vom Benutzer eingesendeter Kommentar, ein interaktives Widget oder Gadget oder ein anderer unabhängiger Inhalt. |
| {{HTMLElement("aside")}}                                                                                                                                                                                                                                                                                                                                                                                                | Repräsentiert einen Teil eines Dokuments, dessen Inhalt nur indirekt mit dem Hauptinhalt des Dokuments in Verbindung steht. Seitenleisten sind häufig als Seitenleisten oder Hinweisboxen dargestellt.                                                                                                                                                                           |
| {{HTMLElement("footer")}}                                                                                                                                                                                                                                                                                                                                                                                               | Repräsentiert einen Footer für sein nächstgelegenes übergeordnetes [Inhaltselement](/de/docs/Web/HTML/Guides/Content_categories#sectioning_content) oder [Wurzelabschnitt](/de/docs/Web/HTML/Reference/Elements/Heading_Elements)-Element. Ein `<footer>` enthält typischerweise Informationen über den Autor des Abschnitts, Copyright-Daten oder Links zu verwandten Dokumenten.                        |
| {{HTMLElement("header")}}                                                                                                                                                                                                                                                                                                                                                                                               | Repräsentiert Einführungsinhalt, typischerweise eine Gruppe von Einführungs- oder Navigationselementen. Es kann einige Überschriftselemente enthalten, aber auch ein Logo, ein Suchformular, einen Autorennamen und andere Elemente.                                                                                                                                               |
| [`<h1>`](/de/docs/Web/HTML/Reference/Elements/Heading_Elements), [`<h2>`](/de/docs/Web/HTML/Reference/Elements/Heading_Elements), [`<h3>`](/de/docs/Web/HTML/Reference/Elements/Heading_Elements), [`<h4>`](/de/docs/Web/HTML/Reference/Elements/Heading_Elements), [`<h5>`](/de/docs/Web/HTML/Reference/Elements/Heading_Elements), [`<h6>`](/de/docs/Web/HTML/Reference/Elements/Heading_Elements) | Repräsentieren sechs Ebenen von Abschnittsüberschriften. `<h1>` ist die höchste Abschnittsebene und `<h6>` ist die niedrigste.                                                                                                                                                                                                                                                 |
| {{HTMLElement("hgroup")}}                                                                                                                                                                                                                                                                                                                                                                                               | Repräsentiert eine Überschrift, gruppiert mit sekundärem Inhalt, wie Unterüberschriften, einem alternativen Titel oder einem Slogan.                                                                                                                                                                                                                                            |
| {{HTMLElement("main")}}                                                                                                                                                                                                                                                                                                                                                                                                 | Repräsentiert den dominanten Inhalt des Hauptteils eines Dokuments. Der Hauptinhaltsbereich besteht aus Inhalten, die direkt mit dem zentralen Thema eines Dokuments oder der zentralen Funktionalität einer Anwendung in Verbindung stehen oder diese erweitern.                                                                                                               |
| {{HTMLElement("nav")}}                                                                                                                                                                                                                                                                                                                                                                                                  | Repräsentiert einen Abschnitt einer Seite, dessen Zweck es ist, Navigationslinks bereitzustellen, entweder innerhalb des aktuellen Dokuments oder zu anderen Dokumenten. Häufige Beispiele für Navigationsabschnitte sind Menüs, Inhaltsverzeichnisse und Indizes.                                                                                                                                                    |
| {{HTMLElement("section")}}                                                                                                                                                                                                                                                                                                                                                                                              | Repräsentiert einen generischen eigenständigen Abschnitt eines Dokuments, der kein spezifischeres semantisches Element zu seiner Darstellung hat. Abschnitte sollten immer eine Überschrift haben, mit nur sehr wenigen Ausnahmen.                                                                                                                                                                                    |
| {{HTMLElement("search")}}                                                                                                                                                                                                                                                                                                                                                                                               | Repräsentiert einen Teil, der einen Satz von Formularelementen oder anderen Inhalten umfasst, die mit der Durchführung einer Such- oder Filteroperation in Verbindung stehen.                                                                                                                                                                                                                                           |

## Textinhalt

Verwenden Sie HTML-Textinhaltselemente, um Blöcke oder Abschnitte von Inhalten zu organisieren, die zwischen den öffnenden {{HTMLElement("body")}} und schließenden `</body>` Tags platziert sind. Wichtig für {{Glossary("accessibility", "Barrierefreiheit")}} und {{Glossary("SEO", "SEO")}} identifizieren diese Elemente den Zweck oder die Struktur dieses Inhalts.

| Element                       | Beschreibung                                                                                                                                                                                                                                                                                                                                                                                           |
| ----------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| {{HTMLElement("blockquote")}} | Gibt an, dass der eingeschlossene Text ein längeres Zitat ist. Üblicherweise wird dies visuell durch Einrückungen dargestellt. Eine URL für die Quelle des Zitats kann über das `cite`-Attribut angegeben werden, während eine Textdarstellung der Quelle mithilfe des {{HTMLElement("cite")}}-Elements gegeben werden kann.                                                                                                          |
| {{HTMLElement("dd")}}         | Liefert die Beschreibung, Definition oder den Wert für den vorhergehenden Begriff ({{HTMLElement("dt")}}) in einer Definitionsliste ({{HTMLElement("dl")}}).                                                                                                                                                                                                                                          |
| {{HTMLElement("div")}}        | Der generische Container für Flussinhalt. Es hat keine Auswirkungen auf den Inhalt oder das Layout, bis es auf irgendeine Weise mit CSS gestylt wird (z.B. wird Styling direkt darauf angewendet, oder es wird ein Layout-Modell, wie {{Glossary("Flexbox", "Flexbox")}}, auf dessen übergeordnetes Element angewendet).                                                                                     |
| {{HTMLElement("dl")}}         | Repräsentiert eine Definitionsliste. Das Element umschließt eine Liste von Gruppen von Begriffen (spezifiziert mit dem {{HTMLElement("dt")}}-Element) und Beschreibungen (bereitgestellt durch {{HTMLElement("dd")}}-Elemente). Häufige Verwendungen dieses Elements sind die Implementierung eines Glossars oder die Anzeige von Metadaten (einer Liste von Schlüssel-Wert-Paaren).                                                          |
| {{HTMLElement("dt")}}         | Spezifiziert einen Begriff in einer Definitions- oder Beschreibungs-Liste und muss daher innerhalb eines {{HTMLElement("dl")}}-Elements verwendet werden. Es wird normalerweise von einem {{HTMLElement("dd")}}-Element gefolgt; jedoch weisen mehrere `<dt>`-Elemente in einer Reihe darauf hin, dass mehrere Begriffe alle durch das unmittelbar darauf folgende {{HTMLElement("dd")}}-Element definiert werden.                                       |
| {{HTMLElement("figcaption")}} | Repräsentiert eine Bildunterschrift oder Legende, die den Rest des Inhalts ihres übergeordneten {{HTMLElement("figure")}}-Elements beschreibt.                                                                                                                                                                                                                                                         |
| {{HTMLElement("figure")}}     | Repräsentiert eigenständigen Inhalt, möglicherweise mit einer optionalen Bildunterschrift, die mithilfe des {{HTMLElement("figcaption")}}-Elements angegeben wird. Die Figur, ihre Bildunterschrift und ihr Inhalt werden als Einheit referenziert.                                                                                                                                                     |
| {{HTMLElement("hr")}}         | Repräsentiert einen thematischen Bruch zwischen Elemente auf Absatzebene: zum Beispiel eine Szenenwechsel in einer Geschichte oder ein Themenwechsel innerhalb eines Abschnitts.                                                                                                                                                                                                                      |
| {{HTMLElement("li")}}         | Repräsentiert ein Element in einer Liste. Es muss in einem übergeordneten Element enthalten sein: einer geordneten Liste ({{HTMLElement("ol")}}), einer ungeordneten Liste ({{HTMLElement("ul")}}) oder einem Menü ({{HTMLElement("menu")}}). In Menüs und ungeordneten Listen werden Listenelemente normalerweise als Aufzählungspunkte dargestellt. In geordneten Listen werden sie üblicherweise mit einem aufsteigenden Zähler links angezeigt, wie z. B. eine Zahl oder ein Buchstabe. |
| {{HTMLElement("menu")}}       | Eine semantische Alternative zu {{HTMLElement("ul")}}, wird jedoch von Browsern (und durch den Barrierefreiheit-Baum) nicht anders als {{HTMLElement("ul")}} behandelt. Es repräsentiert eine ungeordnete Liste von Elementen (die durch {{HTMLElement("li")}}-Elemente repräsentiert werden).                                                                                                                                                 |
| {{HTMLElement("ol")}}         | Repräsentiert eine geordnete Liste von Elementen — üblicherweise als nummerierte Liste dargestellt.                                                                                                                                                                                                                                                                                                      |
| {{HTMLElement("p")}}          | Repräsentiert einen Absatz. Absätze werden normalerweise im visuellen Medium als Textblöcke dargestellt, die durch Leerzeilen und/oder eine Erste-Zeilen-Einrückung von angrenzenden Blöcken getrennt sind, aber HTML-Absätze können jede strukturelle Gruppierung von verwandtem Inhalt sein, wie Bilder oder Formulare.                                                                                                                   |
| {{HTMLElement("pre")}}        | Repräsentiert vorformatierten Text, der genau so dargestellt werden soll, wie er in der HTML-Datei geschrieben ist. Der Text wird normalerweise unter Verwendung einer nicht-proportionalen oder [monospaced](https://en.wikipedia.org/wiki/Monospaced_font) Schriftart gerendert. Leerzeichen innerhalb dieses Elements werden wie geschrieben angezeigt.                                                                       |
| {{HTMLElement("ul")}}         | Repräsentiert eine ungeordnete Liste von Elementen, typischerweise als eine Aufzählungsliste dargestellt.                                                                                                                                                                                                                                                                                               |

## Inline-Textsemantik

Verwenden Sie die HTML-Inline-Textsemantik, um die Bedeutung, Struktur oder den Stil eines Wortes, einer Zeile oder eines beliebigen Textstücks zu definieren.

| Element                   | Beschreibung                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                            |
| ------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| {{HTMLElement("a")}}      | Erstellt zusammen mit seinem `href`-Attribut einen Hyperlink zu Webseiten, Dateien, E-Mail-Adressen, Orten innerhalb der aktuellen Seite oder allem anderen, auf das eine URL verweisen kann.                                                                                                                                                                                                                                                                                                                                                          |
| {{HTMLElement("abbr")}}   | Repräsentiert eine Abkürzung oder ein Akronym.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                         |
| {{HTMLElement("b")}}      | Wird verwendet, um die Aufmerksamkeit des Lesers auf den Inhalt des Elements zu lenken, dem ansonsten keine besondere Bedeutung zukommt. Dies war früher bekannt als das Boldface-Element, und die meisten Browser zeigen den Text weiterhin fett an. Sie sollten `<b>` jedoch nicht zum Stylen von Text oder zur Hervorhebung von Bedeutung verwenden. Wenn Sie fetten Text erstellen möchten, sollten Sie die CSS {{cssxref("font-weight")}}-Eigenschaft verwenden. Wenn Sie angeben möchten, dass ein Element von besonderer Bedeutung ist, sollten Sie das {{HTMLElement("strong")}}-Element verwenden.                        |
| {{HTMLElement("bdi")}}    | Weist den bidirektionalen Algorithmus des Browsers an, den enthaltenen Text in Isolation von seinem umgebenden Text zu behandeln. Dies ist besonders nützlich, wenn eine Website dynamisch Text einfügt und die Richtung des eingefügten Textes nicht kennt.                                                                                                                                                                                                                                                                                            |
| {{HTMLElement("bdo")}}    | Überschreibt die aktuelle Textausrichtungsrichtung, sodass der Text innerhalb in einer anderen Richtung gerendert wird.                                                                                                                                                                                                                                                                                                                                                                                                                               |
| {{HTMLElement("br")}}     | Erzeugt einen Zeilenumbruch im Text (Zeilenumbruch). Es ist nützlich, um ein Gedicht oder eine Adresse zu schreiben, bei denen die Aufteilung der Zeilen signifikant ist.                                                                                                                                                                                                                                                                                                                                                                             |
| {{HTMLElement("cite")}}   | Wird verwendet, um den Titel eines kreativen Werkes zu kennzeichnen. Der Verweis kann abhängig von kontextbezogenen Konventionen, die sich auf Zitatenmetadaten beziehen, in einer abgekürzten Form vorliegen.                                                                                                                                                                                                                                                                                                                                        |
| {{HTMLElement("code")}}   | Zeigt seinen Inhalt so gestylt an, dass der Eindruck entsteht, der Text sei ein kurzer Ausschnitt von einem Computerprogramm. Standardmäßig wird der Textinhalt mit der Standardschriftart der Benutzeragenten für Monospace angezeigt.                                                                                                                                                                                                                                                                                                                   |
| {{HTMLElement("data")}}   | Verknüpft ein gegebenes Inhaltselement mit einer maschinenlesbaren Übersetzung. Wenn die enthaltenen Informationen zeit- oder datumsbezogen sind, muss das `<time>`-Element verwendet werden.                                                                                                                                                                                                                                                                                                                                                          |
| {{HTMLElement("dfn")}}    | Ist dazu bestimmt, den in Verbindung mit einem Definitionstext oder -satz definierten Begriff anzuzeigen. Das übergeordnete {{HTMLElement("p")}}-Element, das {{HTMLElement("dt")}}/{{HTMLElement("dd")}}-Paar oder der nächste übergeordnete Abschnittsvorfahr des `<dfn>`-Elements wird als Definition des Begriffs betrachtet.                                                                                                                                                                                                                           |
| {{HTMLElement("em")}}     | Markiert Text, der Betonungsakzent enthält. Das `<em>`-Element kann verschachtelt werden, wobei jede Verschachtelung ein höheres Maß an Betonung angibt.                                                                                                                                                                                                                                                                                                                                                                                                |
| {{HTMLElement("i")}}      | Stellt einen Textbereich dar, der aus irgendeinem Grund vom normalen Text abgesetzt ist, wie beispielsweise idiomatische Ausdrücke, technische Begriffe und taxonomische Bezeichnungen. Historisch gesehen wurden diese mithilfe von kursive Schrift angezeigt, wodurch das `<i>`-Element seinen Namen erhielt.                                                                                                                                                                                                                                        |
| {{HTMLElement("kbd")}}    | Repräsentiert einen Bereich von Inline-Text, der textuelle Nutzereingaben von einer Tastatur, Spracheingabe oder einem anderen Texteingabegerät angibt. Üblicherweise rendert der Benutzeragent den Inhalt eines `<kbd>`-Elements standardmäßig in seiner eigenen Monospace-Schriftart, obwohl dies nicht durch den HTML-Standard vorgeschrieben ist.                                                                                                                                                                                                        |
| {{HTMLElement("mark")}}   | Repräsentiert Text, der markiert oder hervorgehoben wurde, um ihn bezugnehmend oder annotierend aufgrund seiner Relevanz im umgebenden Kontext zu verwenden.                                                                                                                                                                                                                                                                                                                                                                                        |
| {{HTMLElement("q")}}      | Gibt an, dass der eingeschlossene Text ein kurzes Inline-Zitat ist. Die meisten modernen Browser implementieren dies, indem sie den Text in Anführungszeichen setzen. Dieses Element ist für kurze Zitate gedacht, die keine Absatzumbrüche erfordern; für lange Zitate verwenden Sie das {{HTMLElement("blockquote")}}-Element.                                                                                                                                                                                                                     |
| {{HTMLElement("rp")}}     | Wird verwendet, um Backup-Klammern in Browsern bereitzustellen, die die Anzeige von Ruby-Anmerkungen mithilfe des {{HTMLElement("ruby")}}-Elements nicht unterstützen. Ein `<rp>`-Element sollte jede der öffnenden und schließenden Klammern, die das {{HTMLElement("rt")}}-Element umschließen, enthalten, das den Text der Anmerkung enthält.                                                                                                                                                                                                  |
| {{HTMLElement("rt")}}     | Gibt die Ruby-Komponente der Ruby-Anmerkung an, die dazu verwendet wird, Aussprache-, Übersetzungs- oder Transliterationsinformationen für ostasiatische Typografie bereitzustellen. Das `<rt>`-Element muss immer innerhalb eines {{HTMLElement("ruby")}}-Elements enthalten sein.                                                                                                                                                                                                                                                                     |
| {{HTMLElement("ruby")}}   | Repräsentiert kleine Anmerkungen, die oberhalb, unterhalb oder neben Basistext gerendert werden, meist verwendet, um die Aussprache von ostasiatischen Zeichen zu zeigen. Es kann auch verwendet werden, andere Arten von Texten zu kommentieren, allerdings ist dies weniger üblich.                                                                                                                                                                                                                                                             |
| {{HTMLElement("s")}}      | Rendert Text mit einem Durchstreichungseffekt oder einer Linie darüber. Verwenden Sie das `<s>`-Element, um Dinge darzustellen, die nicht mehr relevant oder nicht mehr zutreffend sind. `<s>` ist jedoch nicht dafür geeignet, um Dokumentänderungen anzuzeigen; hierfür sollten Sie die Elemente {{HTMLElement("del")}} und {{HTMLElement("ins")}} verwenden, je nachdem, was zutreffend ist.                                                                                                                                                      |
| {{HTMLElement("samp")}}   | Wird verwendet, um Inline-Text zu umhüllen, der Beispiel- (oder zitierten) Ausgabe eines Computerprogramms darstellt. Dessen Inhalt wird üblicherweise mit der Standardschriftart des Browsers für Monospace-Schriftarten gerendert (wie [Courier](<https://en.wikipedia.org/wiki/Courier_(typeface)>) oder Lucida Console).                                                                                                                                                                                                                        |
| {{HTMLElement("small")}}  | Repräsentiert Kommentare und Kleindruck, wie Copyright- und Rechtstexte, unabhängig von deren stilistischer Präsentation. Standardmäßig rendert es Text innerhalb einer Schriftgröße kleiner, wie von `small` zu `x-small`.                                                                                                                                                                                                                                                                                |
| {{HTMLElement("span")}}   | Ein generischer Inline-Container für Phraseninhalt, der an sich nichts repräsentiert. Er kann verwendet werden, um Elemente für Stylingzwecke zu gruppieren (unter Verwendung der `class`- oder `id`-Attribute) oder weil sie gemeinsame Attributwerte, wie `lang`, teilen. Es sollte nur verwendet werden, wenn kein anderes semantisches Element geeignet ist. `<span>` ist einem Div-Element sehr ähnlich, aber ein Div ist ein {{Glossary("Block-level_content", "Block-Level-Element")}}, während ein `<span>` ein {{Glossary("Inline-level_content", "Inline-Level-Element")}} ist. |
| {{HTMLElement("strong")}} | Gibt an, dass sein Inhalt von großer Wichtigkeit, Ernsthaftigkeit oder Dringlichkeit ist. Browser stellen den Inhalt üblicherweise in Fettschrift dar.                                                                                                                                                                                                                                                                                                                                                                                              |
| {{HTMLElement("sub")}}    | Gibt Inline-Text an, der aus rein typografischen Gründen als Tiefgestellt angezeigt werden soll. Tiefstellungen werden typischerweise mit einer verringerten Grundlinie unter Verwendung einer kleineren Schriftgröße gerendert.                                                                                                                                                                                                                                                                                                                    |
| {{HTMLElement("sup")}}    | Gibt Inline-Text an, der aus rein typografischen Gründen als Hochgestellt angezeigt werden soll. Hochstellungen werden typischerweise mit einer erhöhten Grundlinie unter Verwendung einer kleineren Schriftgröße gerendert.                                                                                                                                                                                                                                                                                                                         |
| {{HTMLElement("time")}}   | Repräsentiert einen bestimmten Zeitraum. Es kann das `datetime`-Attribut enthalten, um Daten in maschinenlesbarem Format zu übersetzen, was bessere Suchmaschinenergebnisse oder benutzerdefinierte Funktionen wie Erinnerungen ermöglicht.                                                                                                                                                                                                                                                                                                          |
| {{HTMLElement("u")}}      | Repräsentiert einen Bereich von Inline-Text, der in einer Weise gerendert werden soll, die darauf hinweist, dass es eine nicht-textuelle Anmerkung hat. Dies wird standardmäßig als einzelne solide Unterstreichung gerendert, kann jedoch mit CSS geändert werden.                                                                                                                                                                                                                                                                                   |
| {{HTMLElement("var")}}    | Repräsentiert den Namen einer Variablen in einem mathematischen Ausdruck oder einem Programmierkontext. Es wird typischerweise unter Verwendung einer kursivierten Version der aktuellen Schriftart dargestellt, obwohl dieses Verhalten browserabhängig ist.                                                                                                                                                                                                                                                                                         |
| {{HTMLElement("wbr")}}    | Repräsentiert eine mögliche Worttrennung—eine Position innerhalb eines Textes, an der der Browser optional eine Zeile trennen kann, obwohl seine Zeilenumbruchregeln an dieser Stelle ansonsten keinen Umbruch erzeugen würden.                                                                                                                                                                                                                                                                                                                     |

## Bild und Multimedia

HTML unterstützt verschiedene Multimedia-Ressourcen wie Bilder, Audio und Video.

| Element                  | Beschreibung                                                                                                                                                                                                                                                                               |
| ------------------------ | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| {{HTMLElement("area")}}  | Definiert einen Bereich innerhalb einer Bildkarte, die vordefinierte anklickbare Bereiche hat. Eine _Bildkarte_ ermöglicht es, geometrische Bereiche auf einem Bild mit einem {{Glossary("Hyperlink", "Hyperlink")}} zu verknüpfen.                                                                                          |
| {{HTMLElement("audio")}} | Wird verwendet, um Audioinhalte in Dokumenten einzubetten. Es können eine oder mehrere Audioquellen enthalten sein, die mit dem `src`-Attribut oder dem `source`-Element dargestellt werden: Der Browser wählt die geeignetste aus. Es kann auch das Ziel für gestreamte Medien sein, unter Verwendung eines [`MediaStream`](/de/docs/Web/API/MediaStream). |
| {{HTMLElement("img")}}   | Betten Sie ein Bild in das Dokument ein.                                                                                                                                                                                                                                                  |
| {{HTMLElement("map")}}   | Wird mit {{HTMLElement("area")}}-Elementen verwendet, um eine Bildkarte zu definieren (einen anklickbaren Bereich).                                                                                                                                                                        |
| {{HTMLElement("track")}} | Wird als Kindelement der Medienelemente `audio` und `video` verwendet. Es erlaubt es Ihnen, zeitbasierten Text-Titel (oder zeitbasierte Daten) anzugeben, zum Beispiel, um Untertitel automatisch zu handhaben. Die Titel sind im [WebVTT-Format](/de/docs/Web/API/WebVTT_API) (`.vtt`-Dateien, Web Video Text Tracks) formatiert. |
| {{HTMLElement("video")}} | Betten Sie einen Medienplayer ein, der die Videowiedergabe im Dokument unterstützt. Sie können `<video>` auch für Audioinhalte verwenden, aber das `audio`-Element kann eine angemessenere Nutzererfahrung bieten.                                                                                                               |

| {{HTMLElement("picture")}}     | Enthält null oder mehr {{HTMLElement("source")}}-Elemente und ein {{HTMLElement("img")}}-Element, um alternative Versionen eines Bildes für verschiedene Darstellungs-/Geräteszenarien anzubieten.                                                                                                                                                                                                                                                                                                          |

| {{HTMLElement("noscript")}} | Definiert einen Abschnitt von HTML, der eingefügt werden soll, wenn ein Skripttyp auf der Seite nicht unterstützt wird oder wenn Skripting im Browser derzeit deaktiviert ist.                                                                                                                                               |

## Webkomponenten

Webkomponenten sind eine HTML-bezogene Technologie, die es ermöglicht, benutzerdefinierte Elemente zu erstellen und zu verwenden, als wären es reguläre HTML-Elemente. Zusätzlich können Sie benutzerdefinierte Versionen von Standard-HTML-Elementen erstellen.

| Element                     | Beschreibung                                                                                                                                                                                                                                          |
| --------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| {{HTMLElement("slot")}}     | Teil der [Webkomponenten](/de/docs/Web/API/Web_components)-Technologiesuite, dieses Element ist ein Platzhalter in einer Webkomponente, den Sie mit eigenem Markup füllen können, was es ermöglicht, separate DOM-Bäume zu erstellen und diese zusammen darzustellen. |
| {{HTMLElement("template")}} | Ein Mechanismus zum Halten von HTML, das nicht sofort gerendert werden soll, wenn eine Seite geladen wird, aber möglicherweise während der Laufzeit mit JavaScript instanziiert wird.                                                                                         |
