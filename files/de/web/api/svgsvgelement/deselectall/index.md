Die `deselectAll()`-Methode des [`SVGSVGElement`](/de/docs/Web/API/SVGSVGElement)-Interfaces hebt die Auswahl aller ausgewählten Objekte auf, einschließlich aller Auswahlen von Textstrings und Eingabeleisten.

## Syntax

```js-nolint
deselectAll()
```

### Parameter

Keine.

### Rückgabewert

Keine.
