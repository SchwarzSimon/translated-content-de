{{APIRef("HTML DOM")}}

Die schreibgeschützte Eigenschaft **`form`** des [`HTMLButtonElement`](/de/docs/Web/API/HTMLButtonElement)-Interfaces gibt ein [`HTMLFormElement`](/de/docs/Web/API/HTMLFormElement)-Objekt zurück, das dieses {{htmlelement("button")}} enthält, oder `null`, wenn dieses Button-Element zu keinem Formular gehört.

## Wert

Ein [`HTMLFormElement`](/de/docs/Web/API/HTMLFormElement) oder `null`.
