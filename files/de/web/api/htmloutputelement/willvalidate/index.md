{{APIRef("HTML DOM")}}

Die **`willValidate`** schreibgeschützte Eigenschaft der [`HTMLOutputElement`](/de/docs/Web/API/HTMLOutputElement)-Schnittstelle gibt `false` zurück, da {{HTMLElement("output")}}-Elemente keine Kandidaten für die [Einschränkungsvalidierung](/de/docs/Web/HTML/Guides/Constraint_validation) sind.

## Wert

Der boolesche Wert `false`.
