Die **`role`**-Schreibgeschützte Eigenschaft der [`ElementInternals`](/de/docs/Web/API/ElementInternals)-Schnittstelle gibt die [WAI-ARIA Rolle](/de/docs/Web/Accessibility/ARIA/Reference/Roles) für das Element zurück. Zum Beispiel kann ein Kontrollkästchen [`role="checkbox"`](/de/docs/Web/Accessibility/ARIA/Reference/Roles/checkbox_role) haben. Sie spiegelt das `role`-Attribut wider; sie gibt nicht die implizite ARIA-Rolle des Elements zurück, falls vorhanden, es sei denn, sie wird explizit gesetzt.

## Wert

Ein String, der eine ARIA-Rolle enthält. Eine vollständige Liste der ARIA-Rollen finden Sie auf der [ARIA-Techniken-Seite](/de/docs/Web/Accessibility/ARIA/Guides/Techniques).
