Die schreibgeschützte **`root`**-Eigenschaft des [`FileSystem`](/de/docs/Web/API/FileSystem)-Interfaces gibt ein [`FileSystemDirectoryEntry`](/de/docs/Web/API/FileSystemDirectoryEntry)-Objekt an, das das Stammverzeichnis des Dateisystems repräsentiert und mit der [File and Directory Entries API](/de/docs/Web/API/File_and_Directory_Entries_API) verwendet wird.

## Wert

Ein [`FileSystemDirectoryEntry`](/de/docs/Web/API/FileSystemDirectoryEntry), das das Stammverzeichnis des Dateisystems darstellt.
