{{APIRef("HTML DOM")}}

Die **`willValidate`** schreibgeschützte Eigenschaft der [`HTMLFieldSetElement`](/de/docs/Web/API/HTMLFieldSetElement)-Schnittstelle gibt `false` zurück, da {{HTMLElement("fieldset")}}-Elemente keine Kandidaten für die [Einschränkungsvalidierung](/de/docs/Web/HTML/Guides/Constraint_validation) sind.

## Wert

Der boolesche Wert `false`.
