Die schreibgeschützte **`height`**-Eigenschaft der [`PaintSize`](/de/docs/Web/API/PaintSize)-Schnittstelle gibt die Höhe des Ausgabe-Bitmaps zurück, das der Autor zeichnen soll.

## Wert

Eine Gleitkommazahl.
