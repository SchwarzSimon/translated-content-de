Die **`width`**-Schreibgeschützte Eigenschaft des [`PaintSize`](/de/docs/Web/API/PaintSize)-Interfaces gibt die Breite des Ausgabebitmaps zurück, die der Autor zeichnen sollte.

## Wert

Eine Gleitkommazahl.
