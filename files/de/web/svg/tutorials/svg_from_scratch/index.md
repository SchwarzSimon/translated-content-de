Dieses Tutorial zielt darauf ab, die Interna von SVG zu erklären und ist mit technischen Details gespickt. Wenn Sie einfach nur schöne Bilder zeichnen möchten, finden Sie möglicherweise nützlichere Ressourcen auf [der Dokumentationsseite von Inkscape](https://inkscape.org/learn/). Eine weitere gute Einführung in SVG bietet das [SVG Primer](https://www.w3.org/Graphics/SVG/IG/resources/svgprimer.html) des W3C. Schauen Sie sich auch dieses adventskalender-themenbasierte [SVG-Tutorial](https://svg-tutorial.com/) an, das Sie durch das Codieren von 25 festlichen SVGs führt.
