Das **`<feFuncA>`** [SVG](/de/docs/Web/SVG) Filter-Primitive definiert die Transferfunktion für die Alpha-Komponente der Eingabegrafik seines übergeordneten {{SVGElement("feComponentTransfer")}} Elements.
