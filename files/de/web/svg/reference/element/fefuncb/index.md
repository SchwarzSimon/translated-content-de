Das **`<feFuncB>`** [SVG](/de/docs/Web/SVG)-Filter-Primitive definiert die Transferfunktion für die blaue Komponente der Eingabegrafik des übergeordneten {{SVGElement("feComponentTransfer")}}-Elements.

## DOM-Schnittstelle
