Das Attribut **`v-hanging`** gibt die Ausrichtungskoordinate für {{Glossary("glyph", "Glyphen")}} an, um eine hängende {{Glossary("baseline/typography", "Basislinie")}} zu erreichen. Der Wert ist ein Versatz im Schriftkoordinatensystem, relativ zum glyphenspezifischen {{SVGAttr("vert-origin-x")}}-Attribut.

Dieses Attribut kann mit den folgenden SVG-Elementen verwendet werden:

- {{SVGElement("font-face")}}
