Das **`accent-height`** Attribut definiert die Entfernung vom Ursprung zur Oberseite von Akzentzeichen, gemessen durch eine Entfernung innerhalb des Schriftart-Koordinatensystems.

Sie können dieses Attribut mit den folgenden SVG-Elementen verwenden:

- {{SVGElement("font-face")}}

## font-face

Für {{SVGElement("font-face")}} definiert `accent-height` die Entfernung vom Ursprung zur Oberseite von Akzentzeichen, gemessen durch eine Entfernung innerhalb des Schriftart-Koordinatensystems.
