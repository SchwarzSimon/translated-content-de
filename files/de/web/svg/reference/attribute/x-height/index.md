{{Deprecated_Header}}

Das **`x-height`**-Attribut gibt die Höhe von Kleinbuchstaben-Glyphen im Schriftfontskoordinatensystem an.

Dieses Attribut kann mit den folgenden SVG-Elementen verwendet werden:

- {{SVGElement("font-face")}}
