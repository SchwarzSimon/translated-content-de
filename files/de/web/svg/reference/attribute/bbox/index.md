{{deprecated_header}}

Das **`bbox`**-Attribut definiert die maximale Umrahmung eines Fonts.

Sie können dieses Attribut mit den folgenden SVG-Elementen verwenden:

- {{SVGElement("font-face")}}

- `<string>`
  - : Eine kommaseparierte Liste von genau vier Zahlen, die, in der Reihenfolge, den unteren linken x, unteren linken y, oberen rechten x und oberen rechten y des Umrahmungsrechtecks für den gesamten Font angeben.
