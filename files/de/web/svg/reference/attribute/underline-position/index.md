Das `underline-position` Attribut repräsentiert die ideale vertikale Position der Unterstreichung. Die Unterstreichungsposition wird im Koordinatensystem der Schriftart ausgedrückt.

Sie können dieses Attribut mit den folgenden SVG-Elementen verwenden:

- {{ SVGElement("font-face") }}
