Das **`v-alphabetic`** Attribut definiert den Ausrichtungskoordinaten für {{Glossary("glyph", "Glyphen")}}, um eine {{Glossary("/Baseline/Typography", "alphabetische Grundlinie")}} zu erreichen. Der Wert ist ein Offset im Schriftkoordinatensystem relativ zum glyphen-spezifischen {{SVGAttr("vert-origin-x")}} Attribut.

Sie können dieses Attribut mit den folgenden SVG-Elementen verwenden:

- {{SVGElement("font-face")}}
