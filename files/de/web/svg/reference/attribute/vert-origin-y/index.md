{{Deprecated_Header}}

Das **`vert-origin-y`** Attribut gibt die y-Koordinate im Schriftarten-Koordinatensystem des Ursprungs eines {{Glossary("glyph", "glyph")}} an, die beim Zeichnen von vertikal orientiertem Text verwendet werden soll.

Sie können dieses Attribut mit den folgenden SVG-Elementen verwenden:

- {{SVGElement("font")}}
