{{Deprecated_Header}}

Das **`descent`**-Attribut definiert die maximale unakzentuierte Tiefe der Schriftart.

> [!NOTE]
> Es wurde spezifiziert, um die Syntax und Semantik des veralteten `descent`-Descriptors der {{cssxref("@font-face")}} at-rule, wie in einer frühen Version von CSS 2 definiert, zu teilen.

Dieses Attribut kann bei den folgenden SVG-Elementen verwendet werden:

- {{SVGElement("font-face")}}
