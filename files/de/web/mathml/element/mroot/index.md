Das **`<mroot>`** [MathML](/de/docs/Web/MathML)-Element wird verwendet, um Wurzeln mit einem expliziten Index darzustellen. Es akzeptiert zwei Argumente, was zu der Syntax führt: `<mroot> base index </mroot>`.

## Attribute

Dieses Element akzeptiert die [globalen MathML-Attribute](/de/docs/Web/MathML/Global_attributes).

## Beispiele

```html
<math display="block">
  <mroot>
    <mi>x</mi>
    <mn>3</mn>
  </mroot>
</math>
```
