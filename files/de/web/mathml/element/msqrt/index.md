Das **`<msqrt>`** [MathML](/de/docs/Web/MathML)-Element wird verwendet, um Quadratwurzeln anzuzeigen (es wird kein Index angezeigt). Die Quadratwurzel akzeptiert nur ein Argument, was zu folgendem Syntax führt: `<msqrt> base </msqrt>`.

## Attribute

Dieses Element akzeptiert die [globalen MathML-Attribute](/de/docs/Web/MathML/Global_attributes).

## Beispiele

```html
<math display="block">
  <msqrt>
    <mi>x</mi>
  </msqrt>
</math>
```
