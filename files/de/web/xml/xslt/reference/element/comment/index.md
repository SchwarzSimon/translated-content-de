Das `<xsl:comment>`-Element schreibt einen Kommentar in das Ausgabedokument. Es darf nur Text enthalten.

## Syntax

```xml
<xsl:comment>
  TEMPLATE
</xsl:comment>
```

### Erforderliche Attribute

Keine.

### Optionale Attribute

Keine.

### Typ

Anweisung, erscheint innerhalb einer Vorlage.

## Spezifikationen

XSLT, Abschnitt 7.4.

Unterstützt.
